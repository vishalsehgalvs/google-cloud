# Practitioner Deep Dive Path

This is a separate full-depth revision track.

Use this path if you want strong conceptual mastery across your entire note base, not only exam-minimum coverage.

## How this differs from the super crisp path

- Super crisp path: fastest exam-focused revision.
- Deep dive path: full topic understanding with sequence, drills, and checkpoints.

## Suggested usage

1. Read `01_exam_scope_and_strategy.md`.
2. Read `02_full_topic_matrix.md` to understand what to cover and why.
3. Study domain files `03` to `06` in order.
4. Run drills in `07_hands_on_and_scenario_drills.md`.
5. Follow `08_21_day_deep_dive_schedule.md`.

For a blended day-by-day control plan across both tracks, use `../study_dashboard.md`.

## Study mode options

- 7-day mode: compress by combining two days into one.
- 14-day mode: medium pace.
- 21-day mode: full depth (recommended for this path).
