# ACE Refactor Pack - Exam Ready

This pack refactors your full study base into concise, scenario-first notes.

It is built for real exam behavior:

- decision under ambiguity
- traps between similar options
- least-ops and least-privilege choices
- practical command memory

## What Is Included

1. Domain notes with direct service-choice logic
2. Pinpoint trap matrix
3. Real-world scenario drills with wrong-option elimination
4. Command playbook for high-frequency gcloud, gsutil, bq, kubectl patterns
5. Full coverage map for source notes 00 to 148

## How To Use (Fast)

1. Read one domain file per day from 01 to 07.
2. After each domain, read 08_pinpoint_traps.md.
3. Solve 3 to 5 blocks from 09_real_world_scenarios.md.
4. Rehearse related commands from 10_command_playbook.md.
5. Track weak patterns in one line: trigger -> best answer.

## Exam Decision Stack

Use this order when options look similar:

1. Requirement fit first
2. Security and least privilege second
3. Operations burden third
4. Cost and scalability fourth

## Elimination Method (10 seconds)

1. Remove options needing unnecessary manual operations.
2. Remove over-privileged IAM answers.
3. Remove products mismatched to workload type.
4. Choose the most managed answer that still meets explicit constraints.
